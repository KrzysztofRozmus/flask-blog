Installation
------------


1. Install Python

    https://www.python.org/downloads/


2. Download github repository

    ```
    git clone https://github.com/KrzysztofRozmus/flask-blog.git
    ```


3. Create built-in virtual environment

    ```
    python3 -m venv venv
    ```


4. Activate created virtual environment

    * Linux

        ```
        source venv/bin/activate
        ```

    * Windows

        ```
        .\venv\Scripts\activate
        ```


5. Install dependencies

    ```
    pip install -r requirements.txt
    ```


6. Run flask local server

    ```
    python3 run.py
    ```


7. Paste displayed URL address `http://127.0.0.1:5000/` to your browser
